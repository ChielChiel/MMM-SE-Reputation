MMM-SE-Reputation

A PHP library for using the mysql framwork for .csv management.

Setup

Put the config.php file in the root directory of your webserver. After that you can put the MMM-SE-Reputation.php file anywhere in your directory. As long as you include it right it will be accesible from anywhere on your webserver. You can simple use the sql language for .csv files.
